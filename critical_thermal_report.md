## 🔧 深度诊断步骤

### 1. SMC重置 (优先执行)
```
1. 完全关机
2. 按住 Shift + Control + Option + 电源键 10秒
3. 松开所有按键
4. 正常开机
5. 观察温度是否改善
```

### 2. 风扇诊断
```bash
# 检查风扇状态
sudo powermetrics --samplers smc -n 1 | grep -i fan

# 手动设置风扇最大转速（临时）
sudo smc -k F0Tg -w 00ff  # 设置风扇最大转速
```

### 3. 热传感器检查
```bash
# 检查所有热传感器
sudo powermetrics --samplers smc -n 1 | grep -i temp

# 检查CPU温度历史
sudo log show --predicate 'eventMessage contains "CPU" AND eventMessage contains "temp"' --last 1h
```

## 🛠️ 硬件修复方案

### 立即检查项目
1. **散热硅脂**
   - CPU/GPU硅脂可能老化干裂
   - 建议更换高品质硅脂（如Arctic MX-4）
   - 需要专业拆机，建议送修

2. **风扇故障**
   - 0 RPM读数异常，可能风扇损坏
   - 检查风扇连接线
   - 可能需要更换风扇组件

3. **散热片堵塞**
   - 多年使用后散热片可能积满灰尘
   - 需要拆机深度清洁

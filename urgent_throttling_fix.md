## 🔧 系统级修复步骤

### 步骤1: SMC重置 (2分钟)
```
1. 关机
2. 按住 Shift + Control + Option + 电源键 10秒
3. 松开所有按键
4. 开机
```

### 步骤2: NVRAM重置 (1分钟)
```
1. 开机立即按住 Command + Option + P + R
2. 保持20秒，听到第二次启动声
3. 松开按键
```

### 步骤3: 安全模式诊断 (5分钟)
```
1. 开机立即按住 Shift 键
2. 进入安全模式后观察频率
3. 正常重启
```

## 🛠️ 硬件修复方案

### 散热系统检查
1. **清理风扇** (需要拆机)
   - 断开电源和电池
   - 打开后盖
   - 用压缩空气清理风扇和散热片
   - 检查风扇是否转动顺畅

2. **更换散热硅脂** (推荐)
   - CPU和GPU硅脂可能老化
   - 建议每2-3年更换一次
   - 使用高品质硅脂(如Arctic MX-4)

### 电源系统检查
1. **电池健康度**
   ```bash
   system_profiler SPPowerDataType | grep -i "condition"
   ```

2. **电源适配器**
   - 检查功率是否足够(至少85W)
   - 尝试更换适配器测试

### 日常使用
1. **散热管理**

   - 关闭不必要的后台应用
